# Nomad Tracker

Персональный трекер иммиграционных условий с Telegram-уведомлениями.
**Полностью бесплатно** — Groq (LLM) + DuckDuckGo (поиск), без кредитной карты.

---

## Запуск за 3 шага

### Шаг 1 — Установить зависимости
Открой Терминал, перейди в папку проекта и выполни:
```
cd nomad-tracker
.venv/bin/pip install -r requirements.txt
```

### Шаг 2 — Создать .env и вставить ключи
```
cp .env.example .env
```
Открой файл `.env` в TextEdit и заполни:

**GROQ_API_KEY** — бесплатный ключ:
1. Зайди на https://console.groq.com
2. Зарегистрируйся (только email, без карты)
3. API Keys → Create API Key → скопируй

**TELEGRAM_BOT_TOKEN** — токен бота:
1. Открой Telegram → найди @BotFather
2. Напиши `/newbot`, придумай имя боту
3. Скопируй токен вида `123456789:AAF...`

### Шаг 3 — Запустить
```
.venv/bin/python main.py
```
Открой в браузере: **http://localhost:8000**

---

## Настройка Telegram Chat ID

1. Напиши своему боту любое сообщение (например, «привет»)
2. Открой в браузере:
   `https://api.telegram.org/bot<ТОКЕН>/getUpdates`
3. Найди `"id"` внутри `"chat"` — это твой Chat ID
4. Вставь его в Настройки → Telegram на сайте

---

## Как работает

- Каждые 12 часов (настраивается) приложение запускает проверку
- DuckDuckGo ищет свежие новости по каждой стране
- Groq (Llama 3.1 70B) анализирует, важно ли это для твоего профиля
- Если изменения — приходит сообщение в Telegram
- Вся история хранится на странице /alerts
